# WMS Hiring Challenge

**Beat Claude to get an interview.**

Holman Logistics is hiring a WMS Implementation Lead. We believe the 20% of implementation work that matters can't be done by AI. This challenge is designed to prove it.

## How It Works

1. Read the [challenge brief](challenges/wms-implementation-012/brief.md)
2. Submit your answer
3. We compare it blind to Claude's baseline answer
4. If you beat Claude, we want to talk

## Why This Exists

AI can name every EDI transaction, outline a cutover plan, and fill in a scoring rubric. What AI can't do is run a cutover weekend at 2am when the 940s are failing and the commercial team is already in your inbox about a promised go-live. We're hiring for that gap.

If an AI can match you on this challenge, you're competing with a $20/month tool.

## The Rules

Use any tools you want. AI, search, colleagues, whatever.

Claude's baseline answer was generated using the same brief. If you paste the brief into an AI, you'll get a similar answer to the one we already have. **Ties don't advance.**

To beat Claude, you need to add something it can't: real implementation scar tissue, a commercial pushback story, a cutover-weekend ritual, an integration failure you've personally diagnosed.

**Time commitment:** 1-2 hours. Submissions are capped at 2 pages (tables and timelines don't count).
